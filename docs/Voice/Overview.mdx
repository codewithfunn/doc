# API Overview

The **Voice API** allows you to send voice messages using text-to-speech with live and machine call detection. Below are details for the endpoint, headers, request body, and response.

---

## Endpoint

<CodeBlock language="bash">
https://api.intelsend.com/send-msg
</CodeBlock>

---

## Features

- Single communication packet per request.
- Sandbox mode for testing.
- Flexible scheduling with `sendAt`.
