# Request Body

The request body defines the parameters required to send a voice message via the IntelSend API.

---

## Parameters

| Name          | Type               | Required | Sample Value                                                                                                                      | Description                                                                                                                                    |
| ------------- | ------------------ | -------- | --------------------------------------------------------------------------------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------- |
| `template-id` | String             | ‚úÖ Yes   | `688b634bf19b87633060398f`                                                                                                        | value of the template set via IntelSend admin portal.                                                                                          |
| `campaignId`  | String             | ‚ùå No    | `kdjkadslfjl12323`                                                                                                                | value of the Campaign set via IntelSend admin portal.                                                                                          |
| `channel`     | String             | ‚úÖ Yes   | `sms`                                                                                                                             | channel must be sms in case of sending sms campaign                                                                                            |
| `params`      | Array of key-value | ‚úÖ Yes   | `[ { "uid": "abc-123", "to": "abc@gmail.com", "name": "Deepak" }, { "uid": "xyz-123", "to": "xyz@outlook.com", "name": "Jim" } ]` | Array of objects with key-value pairs. Keys are template variable names; values replace them dynamically. If no variables, array can be empty. |
| `to`          | String             | ‚úÖ Yes   | `3233933379`                                                                                                                      | The **sender phone **. Must be assigned/approved.                                                                                              |
| `from`        | String             | ‚úÖ Yes   | `9498671145`                                                                                                                      | The **sender phone**. Must be assigned/approved.                                                                                               |
| `uid`         | String             | ‚úÖ Yes   | `abc-123`                                                                                                                         | Unique identifier for tracking requests. Limit: 40 characters.                                                                                 |
