# FAQ

This section answers common questions about using the IntelSend API.

---

### Q1: What is the maximum SMS character limit?

<Admonition type="info" title="Answer">
600 characters for plain English, 300 characters for Unicode text.
</Admonition>

---

### Q2: What is the default value of `checkOptOut`?

<Admonition type="info" title="Answer">
The default value is `true`.
</Admonition>
