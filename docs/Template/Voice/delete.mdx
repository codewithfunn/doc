## Delete Voice Template by ID

To delete an Voice template, send a `DELETE` request to the IntelSend API with the **template ID**.

<Admonition type="tip" title="Pro Tip">
  Be careful when deleting ‚Äî once removed, templates cannot be recovered.
</Admonition>

```bash
curl --location --request DELETE 'https://beta.intelsend.com/api/template/?id=<TEMPLATE_ID>' \
--header 'Cookie: __Secure-next-auth.session-token=<YOUR_SESSION_TOKEN>'
```
